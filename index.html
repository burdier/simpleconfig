This script will automatically download a snippet from GitHub and insert it before the last line in your .zshrc file.

What Does the enter_dotnet_dir Function Do?

The enter_dotnet_dir function is designed to enhance your Zsh shell experience when working with .NET projects. Hereâ€™s how it works:

  • Checks if the last command was cd: The function first checks if the previous command was cd. If it was, the function does nothing to avoid repetitive executions.
  • Detects a Git repository: If the current directory contains a Git repository, the function proceeds.
  • Looks for a .NET solution file: The function searches for a .sln file within a depth of 5 directories. If found, it continues.
  • Lists available projects: It lists the directories containing .csproj files and uses fzf to allow you to select which project to navigate to.
  • Handles directory permissions: Before navigating to the selected project directory, it checks for read and execute permissions. If permissions are granted, it navigates to the directory and shows the path using lolcat.
  • Displays informative messages: All key messages (like navigation success or failure to find files) are displayed using lolcat for colorful output.

Step 1: Run the Script Below

Copy and paste the following command into your terminal:

curl -fsSL https://raw.githubusercontent.com/burdier/simpleconfig.github.io/main/dotnet_zsh_script.sh -o /tmp/dotnet_zsh_script.sh && { sed '$d' ~/.zshrc; cat /tmp/dotnet_zsh_script.sh; tail -n 1 ~/.zshrc; } > ~/.zshrc.tmp && mv ~/.zshrc.tmp ~/.zshrc && source ~/.zshrc
